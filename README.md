# synop_map

This script fetches SMHI data and plots it on an interactive map

## Dependencies
-  pip install matplotlib
-  pip install folium
-  pip install pandas
-  pip install numpy
-  pip install pypi-json 
-  pip install requests 
-  pip install selenium
